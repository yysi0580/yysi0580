    ### 1) NewsMoa
    뉴스 OpenAPI를 활용한 뉴스 통합 조회 웹 서비스

    - React 기반 뉴스 목록 / 상세 / 카테고리 조회 화면 구현
    - REST API 연동과 비동기 데이터 처리 경험
    - 사용자 관점에서 로딩, 오류, 데이터 없음 상태를 분리해 UI 반영
    - 프론트엔드 중심으로 실제 데이터 기반 서비스 흐름 구현

### 3) Barotruck

- 백엔드에서는 주문, 결제, 정산, 문서화 중심으로 참여
- 프론트, 관리자 웹, 백엔드가 연결되는 실서비스형 구조 경험

### 4) ObserverSpring
내가 혼자 만든 Minecraft 관찰/모니터링 플랫폼 프로젝트

- Spring Boot 4 + Java 21 기반 백엔드 설계
- JWT 보안 설정, 회원가입/로그인, 실시간 스냅샷 수집 API 구현
- 플레이어 주변 정보를 발밑 / 몸높이 / 머리위 3개 레이어로 조회하는 구조 설계
- 플레이어 스냅샷 수집, 활동 로그 수집/조회, 플레이어 목록/상세 조회 API 구현
- Oracle CLOB JSON을 활용해 `11 x 11 x 3` 레이어 데이터를 렌더링 목적에 맞게 저장하도록 도메인 설계
- 프론트 빌드 결과를 백엔드 정적 리소스로 묶고, SPA 라우팅 포워딩까지 연결

🔗 Repository  
- ObserverSpring: https://github.com/ObserverPLUGIN/observerspring
