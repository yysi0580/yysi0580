## Main Project
### Barotruck
통합 물류 운영 플랫폼 백엔드 프로젝트  
- 주문, 결제, 정산 API 구현
- JWT 기반 인증 및 역할별 접근 제어
- Toss Payments 연동 및 상태 흐름 처리

### NewsMoa
뉴스 OpenAPI 기반 뉴스 통합 조회 웹 서비스  
- React 기반 화면 구현
- 비동기 데이터 처리 및 예외 상태 UI 반영

### Florit
AI 기반 꽃 추천 및 주변 꽃집 검색 서비스  
- 추천 결과 및 상세 화면 구현
- 지도 API 연동 및 사용자 인터랙션 처리
